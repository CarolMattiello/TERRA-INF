-> O TERRAINF consiste num jogo estilo Terraria, no qual o usuário minera até atingir
uma pontuação específica para avançar de fase. Existem minérios bons, que dão pontos e
energia, e minérios ruins, que retiram a energia do usuário.

-> O jogador começa com 3 vidas. Quando perder todas, será levado a uma tela de GAME OVER.

-> Para alcançar o nível 2, são necessários 1000 pontos, e para finalizar o jogo, são necessários mais 2000 pontos.


/// COMO ABRIR O JOGO?

-> Baixe a pasta .zip enviada;

-> Dentro dela, haverão imagens (sprites), necessárias para o carregamento correto do jogo, além de dois arquivos texto
nomeados "nivel1.txt" e "nivel2.txt", estes quais são responsáveis para o funcionamento do mesmo;

-> Arquivo .c com o código fonte do jogo;

-> Arquivo executável .exe para o início direto do jogo.

/// COMO JOGAR?

-> O jogador pode ser movido com as teclas "W A S D" do teclado.

-> Para ativar o modo mineração, aperte a tecla "1". Para minerar, aperte a tecla da direção desejada,
enquanto segura a tecla "ESPAÇO". Caso queira sair do modo mineração, aperte "1" novamente.

-> O jogador só pode se mover para cima com escada. Para colocar uma escada, aperte a tecla "SHIFT_LEFT".

-> Para se mover para baixo, é necessário que tenha uma escada no local, ou que não tenha nenhum bloco abaixo do jogador.

-> Caso o jogador caia mais de 3 blocos, perderá uma vida.

-> Caso sua energia seja igual ou menor a 20, também perderá uma vida, mas sua energia será acrescida de 100.
